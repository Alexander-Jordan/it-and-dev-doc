# 82. Two-way binding refresher


#### Method 1
```svelte
<script>
	let val = '';

	$: console.log(val);

	function setValue(event) {
		val = event.target.value;
	}
</script>

<input type="text" value={val} on:input={setValue}>
```

#### Method 2
```svelte
<script>
	let val = '';

	$: console.log(val);
</script>

<input type="text" bind:value={val}>
```

#### Method 3
```svelte
<script>
	let value = '';

	$: console.log(value);
</script>
<input type="text" bind:value>
```
